Para que o affiliate deeplink funcione perfeitamente será necessário configurar as informações abaixo de cadas programa de afiliado, caso tenha dúvida em alguma das informações abaixo, entre em contato com seu Account Manager.

Actionpay


Infelizmente no caso da Actionpay será necessário solicitar a API Key ao seu Account Manager.

Afiliados B2W


Efetue login no painel do Afiliados, no canto inferior a esquerda existe a ferramenta Gerador de Links, aonde é possível verificar o código do FRANQ no campo "Selecione o site", o FRAQ segue o padrão AFL-00-000000.

Afilio




Efetue login no painel do Afilio, no menu "Minhas Informações", clique em "Gerar Token para API", copie o token e cole no primeiro campo, agora para saber seu AffID e SiteID clique aqui, o AffID é o "Seu ID de usuário" e SiteID o "Seu ID no site".

Cityads


Efetue login no painel do Cityads, clique em API no menu a esquerda, copie o Remote_auth.

Lomadee


Efetue login no painel do Lomadee, acesse "Ferramentas > Códigos de Identificação", caso não exista nenhum Source ID, crie um e adicione no campo acima. (Padrão numérico de 8 dígitos)

Rakuten


Efetue login no painel da Rakuten, no menu "Links > Deep Links", gere um deeplink qualquer, no link gerado existe o parâmetro ID, copie e cole no campo acima.

Zanox


Efetue login no painel da Zanox, na tela inicial do painel o número abaixo do adspace é o AdspaceID, agora clique na conta, no menu "Link e ferramentas > API", será exibido o ConnectID.
